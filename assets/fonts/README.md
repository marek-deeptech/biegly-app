# Fonty IBM Plex Sans

Pliki `IBMPlexSans-*.ttf` (Regular / Bold / Italic / BoldItalic) — rodzina **IBM Plex Sans**,
licencja **SIL Open Font License 1.1** (redystrybucja i osadzanie dozwolone).
Źródło: https://github.com/IBM/plex

Używane serwerowo przez `lib/osint/pdf.ts` (pdfmake) do generowania Analizy OSINT (PDF).
